# Etcha-Sketch Project

## Introduction
The Etcha-Sketch project is a fun and interactive application that mimics the classic Etch A Sketch toy, allowing users to draw on a digital canvas. This project is particularly significant as it serves as an introduction to GUI (Graphical User Interface) development using Tkinter, a standard GUI toolkit in Python.

## Project Overview
In this project, we utilised Tkinter to create a windowed interface where users can manipulate a drawing canvas using mouse events. The application allows for various functionalities such as clearing the canvas, changing colors, and moving four brushes simultaneously to produce mirror art.

## Key Features
- **Drawing on Canvas**: Users can use keyboard arrows to create drawings on the canvas.
- **Screen Selection**: Users can choose different dimensions for their GUI screen size.
- **Clear Functionality**: A button to reset the canvas for new drawings.

## Learning Objectives
This project not only showcases the fun of creating a digital drawing tool but also aims to provide hands-on experience with:
- Event-driven programming
- The architecture of GUI applications
- Using widgets and managing layouts in Tkinter

## Conclusion
Through the Etcha-Sketch project, we explore the endless possibilities offered by GUI development, making programming more accessible and enjoyable!
